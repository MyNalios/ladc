Odoo 14 POS Gift Card (Community)

This module allow user to generate gift cards and customer can use this cards to make payment.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • Here we can use these features by configuration settings.


  • After enable the configuration there is one button, by clicking the Gift Card button there is open new screen, in this screen gift card can be create and list of all created gift card.


  • You can create new gift card by click on create button
